# Space Defense

A simple arcade game created with Processing.

**Team Members:**
- Beshr
- Natalia

## How to Run the Game

1. Download and install Processing from [processing.org](https://processing.org/download)
2. Download the `SpaceDefense.pde` file from this repository
3. Open the file in Processing
4. Click the Play button (▶) to start the game

## Controls

| Key | Action |
|-----|--------|
| ← Left Arrow | Move spaceship left |
| → Right Arrow | Move spaceship right |
| Spacebar | Shoot laser |
| R | Restart game |
